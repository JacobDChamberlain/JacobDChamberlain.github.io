Walk Tracker

Mobile-first web app for measuring walking routes. Built with vanilla JavaScript (ES modules) and Leaflet on free, keyless OpenStreetMap tiles. Tap the map to trace a route, drag points to fine-tune, and get live distance in miles and kilometers (great-circle/haversine math). Features Nominatim place search, browser geolocation, a glowing-path UI with light/dark theming, and a provider-agnostic map layer designed for an easy swap to Google Maps.
